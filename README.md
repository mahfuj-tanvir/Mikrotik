MikroTik Multi-Department Enterprise Network and PPPoE ISP Simulation.

Designed and implemented a virtual multi-department enterprise network using MikroTik Cloud Hosted Router (CHR) on Microsoft Hyper-V. Integrated Windows Server DNS services and simulated an ISP environment using PPPoE to provide secure Internet connectivity, centralized network services, and bandwidth-managed remote access.

Key Implementation:

Designed the network using VLSM and configured WAN connectivity, interface IP addressing, NAT Masquerade, and DHCP Relay.
Integrated Windows Server 2022 DNS services and configured DHCP Servers for departmental networks with DNS suffix distribution.
Deployed a PPPoE Server with Bronze, Silver, and Gold bandwidth profiles, and configured a MikroTik CHR-based PPPoE Client Router with a Home LAN.
Configured router identification using System Note, created router backups, and exported configurations for recovery and maintenance.
Verified end-to-end network functionality using Ping, NSLookup, IPConfig, and Traceroute, and troubleshot DHCP, DNS, NAT, and PPPoE services.

Technologies & Tools: Microsoft Hyper-V, MikroTik RouterOS (CHR), WinBox, Windows Server 2022, VLSM, DHCP, DHCP Relay, DNS, NAT Masquerade, PPPoE, PPP Profiles, IPv4, Network Troubleshooting.
